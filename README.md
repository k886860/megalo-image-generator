# Megalo Image Generator Pro v1.42

Megalo Image Generator is a Chrome extension workflow for generating, organizing, recovering, splitting, and saving images with Google Cloud Vertex AI. It also includes Vertex AI Veo video tools.

> The application is free, but Google Cloud Vertex AI and Cloud Run usage may incur charges on your own account.

## Quick Install

1. Extract the downloaded ZIP completely.
2. Open `chrome://extensions` in Chrome.
3. Enable **Developer mode**.
4. Click **Load unpacked**.
5. Select one language folder's `extension` directory:
   - Korean: `ko_KR/extension`
   - English: `en_US/extension`
   - Japanese: `ja_JP/extension`
   - Simplified Chinese: `zh_CN/extension`
6. Enter your Cloud Run Backend URL and Google Cloud Project ID.

## Safety And Cost

- Start with one reference image, one scene, and one output.
- Confirm your Google Cloud project, model access, quota, and billing before a large batch.
- Use the built-in batch and session budget limits.
- Generation retries can increase usage and cost.
- Follow Google Cloud, Vertex AI, and applicable local policies.
