# megalo-image-generator
Vertex AI image and video generation workflow
Megalo Image Generator Pro v1.42

Free GitHub release of the complete Megalo Image Generator Pro workflow.

## Install

1. Download the protected ZIP from Releases.
2. Extract it completely.
3. Open `chrome://extensions`.
4. Enable Developer mode and choose **Load unpacked**.
5. Select the language-specific `extension` folder.

Languages: Korean, English, Japanese, Simplified Chinese.

Google Cloud Vertex AI and Cloud Run charges are not included.
